An open-source Minecraft launcher written in Python. 
As for why I want to mess with this, it's because I also want to play MC in a familiar way on SteamOS
It is currently very rudimentary, with only the most basic version selection and download (excluding downloading various mods and their loaders, multi-threaded downloads). 
Its GUI is not yet complete
We are able to handle Microsoft account login.
We created this project with the aim of learning Python and Flutter. 
We have relied heavily on the help of AI, and if there are any shortcomings, please understand.
